# Sales Performance & Business Insights Dashboard

## Overview

This project demonstrates advanced Excel skills through the creation of a dynamic sales analysis dashboard.

The dashboard analyzes sales performance across employees, departments, regions, products, and time periods.

## Features

### Data Analysis
- Sales categorization (High, Medium, Low)
- Growth analysis by employee
- Priority classification
- Business KPI reporting

### Advanced Excel Functions
- IF
- OR, AND
- SUMIFS
- COUNTIFS
- Pivot Tables
- INDEX, MATCH
- LET

### Business Insights
- Top-performing employee
- Top-performing region
- Top-selling product
- Highest-performing department

### Dashboard Components
- Sales by Department & Product Type
- Sales by Region
- Sales by Month
- KPI cards
